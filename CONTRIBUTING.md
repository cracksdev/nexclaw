# Contributing

Thanks for your interest in NexClaw.

## Getting started

1. Fork the repository and clone it locally.
2. Follow the [README.md](README.md) **Quick start** in `desktop/` (`npm install`, `npm run dev`).
3. Make changes in focused commits with clear messages.

## Pull requests

- Prefer small PRs that fix one concern.
- Run `npm run build` in `desktop/` before submitting when you touch app code.
- Describe **what** changed and **why** in the PR description.

## Style

- Match existing TypeScript/React patterns in `desktop/renderer` and `desktop/main`.
- Avoid drive-by refactors unrelated to your fix or feature.
